# Dr. Maya Reynolds – Therapy Website

This is a fully responsive therapy website built for **Dr. Maya Reynolds, PsyD**, a Licensed Clinical Psychologist based in Santa Monica, California.

The project was created as part of a frontend design and UI replication assignment.

---

## 🌿 Project Overview

This website is designed to:

- Clearly communicate Dr. Reynolds' specialty in **Anxiety & Trauma Therapy**
- Highlight her **Santa Monica location**
- Reflect a calm, grounded, and professional therapy practice
- Be fully responsive across desktop and mobile devices
- Use a cohesive custom color theme and typography system

---

## 🎯 Key Features

- Fully responsive homepage (desktop, tablet, mobile)
- Custom color palette (replacing original lilac theme)
- SEO-optimized heading structure
- Location-based copywriting
- Services section derived from therapist profile
- About section based on provided bio
- Custom “Our Office” section
- FAQ section
- Strong call-to-action section
- Professional footer with contact details

---

## 🎨 Design System

### Color Palette
- Background: Warm Off-White (#FAF9F6)
- Primary: Deep Green (#2F4A2F)
- Secondary: Soft Neutral Beige
- Accent: Olive CTA section

The theme was intentionally chosen to reflect:
- Calmness
- Trust
- Stability
- Professionalism

### Typography
- Headings: Montserrat
- Body Text: Karla

The fonts were selected to maintain clarity, readability, and modern aesthetic balance.

---

## 🧠 SEO Strategy

- Location-based keywords:
  - “Anxiety & Trauma Therapy in Santa Monica, CA”
- Specialty terms included in headings and body content
- Optimized metadata in layout.tsx
- Clear hierarchy of H1, H2, H3 tags

---

## 🖥️ Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Vercel Deployment
- Google Fonts via next/font
